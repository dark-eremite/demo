# Алгоритм разработки приложения "СтройМатериалы"

## Описание алгоритма

### 1. Начало работы приложения
- Пользователь открывает приложение
- Проверяется авторизация пользователя

### 2. Если пользователь не авторизован
- Отображается страница входа
- Пользователь вводит логин и пароль
- Данные проверяются в базе данных
- Если данные неверны - отображается ошибка
- Если данные верны - пользователь авторизован

### 3. Если пользователь авторизован
- Определяется роль пользователя (клиент, менеджер, администратор)
- Отображается ФИО пользователя в правом верхнем углу

### 4. Просмотр товаров
- Загружается список товаров из базы данных
- Для каждого товара проверяются:
  - Наличие изображения (если нет - показывается заглушка)
  - Размер скидки (>12% - фон #F4A460)
  - Наличие на складе (0 - фон #87CEEB)
  - Цена со скидкой (перечёркнутая красная + новая чёрная)
- Отображаются: фото, наименование, категория, описание, производитель, поставщик, цена, ед.изм., количество, скидка

### 5. Расчёт материалов
- Пользователь выбирает продукт и количество
- Загружается состав продукта из базы данных
- Рассчитывается общее количество материалов
- Отображается результат

### 6. Выход из системы
- Пользователь нажимает "Выйти"
- Сессия завершается
- Возврат на страницу входа

## Блок-схема (описание по ГОСТ 19.701-90)

### Блоки:
- Овал - Начало/Конец
- Прямоугольник - Процесс
- Ромб - Решение
- Параллелограмм - Ввод/Вывод
- Прямоугольник с двойной границей - Предварительная подготовка

### Поток:
1. [Начало] -> 2
2. [Проверка авторизации] -> 3 (да) / 4 (нет)
3. [Определение роли] -> 5
4. [Отображение формы входа] -> 6
5. [Ввод данных] -> 7
6. [Проверка данных] -> 8 (верно) / 4 (неверно)
7. [Авторизация] -> 5
8. [Отображение главной страницы] -> 9
9. [Выбор действия] -> 10 (просмотр товаров) / 11 (расчёт материалов) / 12 (выход)
10. [Загрузка товаров] -> 13
11. [Выбор продукта] -> 14
12. [Выход] -> [Конец]
13. [Проверка каждого товара] -> 10
14. [Расчёт материалов] -> 11

## Примечания

- Все идентификаторы используют snake_case (Python)
- Базовые типы данных соответствуют стандарту
- Цвета подсветки:
  - Скидка >12%: #F4A460 (песочный)
  - Нет на складе: #87CEEB (голубой)
  - Зачёркнутая цена: красный шрифт
  - Новая цена: чёрный шрифт
